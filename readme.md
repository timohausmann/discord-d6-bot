## General Features

* Dices are sorted numerical
* Highest dices are highlighted
* German support (`w` instead of `d`, e.g. `/8w6`)
* Optional `sum` parameter to sum up all dices (this may be handy for some cases or other games), e.g. `/8d6 sum`
